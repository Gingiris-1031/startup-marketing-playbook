## 제로에서 시작하는 SEO

백링크나 도메인 권위 없이 구글 상위 노출하는 방법:

- **롱테일 키워드 타겟팅**: 타겟 사용자가 실제로 검색하는 경쟁이 적은 쿼리를 찾으세요
- **깊이 있는 콘텐츠**: 얕은 글 10개보다 결정판 가이드 1개가 이깁니다
- **테크니컬 SEO 기반**: 사이트 속도, 모바일 퍼스트, 구조화 데이터를 첫날부터 적용

- **내부 링크 설계**: 스마트한 링크 구조로 토피컬 오소리티 구축

## 커뮤니티 침투

기존 커뮤니티에서 스팸 없이 진정한 가치를 제공하는 법:

- **타겟이 모이는 곳 파악**: ICP(이상적 고객)가 어디에 있나요? (Reddit, Discord, Slack, 포럼)
- **가치 제공 우선**: 질문에 답하고, 인사이트를 공유하세요 — 제품부터 꺼내지 마세요
- **시간을 들여 신뢰 구축**: 꾸준한 도움 → 자연스럽게 당신이 뭘 만드는지 궁금해함
- **커뮤니티 네이티브 콘텐츠**: 각 커뮤니티의 톤, 포맷, 기대에 맞추세요
- **전략적 타이밍**: 누군가의 문제에 직접 관련될 때만 제품을 공유하세요

## 소셜 미디어 플레이북

오가닉으로 가입을 유도하는 플랫폼별 콘텐츠 전략:

- **X/Twitter**: 여정을 기록하는 스레드, 전술적 인사이트, 동료와의 소통
- **LinkedIn**: 긴 형식의 소트 리더십, 업계 분석, 역발상 관점
- **Reddit**: 니치 서브레딧에 진심으로 참여, AMA, 케이스 스터디 게시
- **YouTube**: 튜토리얼 콘텐츠, 제품 워크스루, 업계 설명
- **TikTok/Reels**: 퀵 팁, 비포&애프터 데모, 빌딩 비하인드 스토리
