# スタートアップマーケティング（予算ゼロ）— オーガニックチャネル完全ガイド

お金がない？問題ありません。ここで紹介する戦術はすべて無料 — 必要なのはあなたの時間と創造力だけです。

## ゼロからのSEO

バックリンクやドメインオーソリティなしでGoogleで上位表示する方法：

- **ロングテールキーワード戦略**: ターゲットユーザーが実際に検索する競合の少ないクエリを見つける
- **深さ重視のコンテンツ**: 浅い記事10本より、決定版ガイド1本が勝つ
- **テクニカルSEOの基盤**: サイトスピード、モバイルファースト、構造化データを初日から
- **プログラマティックSEO**: テンプレート駆動のページでスケーラブルなオーガニックトラフィックを獲得
- **内部リンク設計**: スマートなリンク構造でトピカルオーソリティを構築

## コミュニティ浸透

既存コミュニティでスパムせずに本当の価値を提供する：

- **ターゲットの居場所を特定**: ICP（理想顧客）はどこにいる？（Reddit、Discord、Slack、フォーラム）
- **価値提供ファースト**: 質問に答え、洞察を共有 — プロダクトから始めない
- **時間をかけて信頼を構築**: 一貫した有益なプレゼンス → あなたが何を作っているか自然に興味を持たれる
- **コミュニティネイティブなコンテンツ**: 各コミュニティのトーン、フォーマット、期待に合わせる
- **戦略的なタイミング**: 誰かの問題に直接関連する場合にのみプロダクトを共有

## ソーシャルメディアプレイブック

オーガニックでサインアップを促すプラットフォーム別コンテンツ戦略：

- **X/Twitter**: ジャーニーを記録するスレッド、戦術的インサイト、ピアとのエンゲージメント
- **LinkedIn**: 長文のソートリーダーシップ、業界分析、逆張りの視点
- **Reddit**: ニッチなサブレディットへの真摯な参加、AMA、ケーススタディ投稿
- **YouTube**: チュートリアルコンテンツ、プロダクトウォークスルー、業界解説
- **TikTok/Reels**: クイックティップス、ビフォーアフターデモ、ビルドの舞台裏

## 口コミエンジニアリング

プロダクトを自然にシェアしたくなる設計：

- **Remarkableな瞬間**: ユーザーが他の人に言わずにいられない「すごい！」体験を設計
- **ソーシャルプルーフフック**: 公開利用指標、バッジ、シェア可能な実績
- **リファラルトリガー**: 他の人を招待するのが自然に感じるプロダクト内の瞬間
- **語りたくなる成果**: ユーザーが自慢したくなる結果を達成させる
- **低フリクションシェア**: ワンクリックシェア、事前作成コピー、ビジュアルアセット

## ゲリラ戦術

コスト以上のインパクトを生むクリエイティブなアプローチ：

- **トレンドに乗る**: ニュースジャッキング、ミームマーケティング、カルチャーモーメント活用
- **競合ポジショニング**: 「Xの代替」ページ、移行ガイド、比較コンテンツ
- **パートナーシップハック**: 競合しない補完プロダクトとのクロスプロモーション
- **ローンチスタッキング**: Product Hunt + HN + Reddit + Twitterを組み合わせて複合効果を狙う
- **予算なしPR**: HARO回答、創業者ストーリーピッチ、データドリブンなプレスアングル
